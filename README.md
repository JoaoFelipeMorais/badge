## Comandos Git
 - git clone (clona um repositório existente)
 - git branch (mostrar a branch atual)
 - git checkout -b (criar uma nova branch)
 - git status (verifica o satus da branch)
 - git commit -m (salva as alterações com mensagem)
 - git push origin (envia as alterações ao repositório remoto)
